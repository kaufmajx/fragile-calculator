A Guide to the Fragile User Interface

Fragile contains a simple, easy-to-use graphical interface that makes inputting mixed fractions, modifying inputs, and viewing results of calculations a breeze. Let's run down a few of the components of the user interface.

The Display:

Fragile contains a display to make seeing mixed fractions easier, but it can be a little daunting if you don't know where to start. Don't worry; the display is much simpler than it looks! The area in the red circle is where stored operands (fractions, whole numbers, mixed numbers) and operations go. Once you enter a fraction and press an operation button or enter 2 fractions and press enter, the current operation and answer will be stored up there. The area in the green circle is where you enter your current operand. Once you are done entering your operand, pressing an operation button or equals will move it into to the red circle area. The hollow black rectangle in the green circle is called a focus indicator. It tells you where your input is currently going. Any inputs you make will be put in the place of the focus indicator.

The Numberpad:

Fragile contains a numberpad with the numbers 0-9 to allow for the simple input of fractions. To use these numbers, simply click on them, and Fragile will add them in their appropriate place.

Operators and Positioning Buttons:

Fragile's user interface also contains buttons for inputting the intended operation, deleting inputs, and clearing the display. Let's go through a few of them now:

Operators: Addition(+), Subtraction(-), Multiplication(X), and Division(÷) buttons allow you to do all common mathematical operations with your inputs. The invert button (Inv) will invert the currently entered item (such as 3/2 becoming 2/3). The simplify button (↡) will simplify a currently entered fraction if possible (such as 1 2/6 becoming 1 1/3). The mediant button (⇹) will compute the mediant of two mixed numbers or fractions, which involves summing the numerators of the two fractions and dividing that by the sum of the two denominators (such as 2 2/3 ⇹ 1 1/2=2 1/5). The integer power button (xⁿ) will raise an input to an integer power (such as 5 xⁿ 2=10). The relational operators (>, <, and ≝) allow for two entries to be compared to determine which is bigger (such as 5 > 2 =True).

Removal: Reset(R), Clear(C), and Backspace(<-) all allow for modification of inputs. Reset will delete the entire current expression. Clear will delete the current mixed fraction. The backspace button will delete the digit to the left of the focus icon. Please note that these modifications are permanent, so take care when using the modification buttons to avoid accidental data loss.

Position: The position button, represented as a box over another box, will move your current input field. For example, if you have just inputted a whole number, and you press the position button, your current input field will now be moved to the numerator of the fraction to the right of the last inputted whole number.

Equals and Sign: The equals("=") button will calculate the result of the currently entered fractions and operators. The sign button (±) will change the sign of the current mixed fraction, from negative to positive and vice versa.

The Toolbar

Fragile contains a toolbar at the top of the user interface which contains multiple useful functions. Let's go over them now.

The toolbar contains six main sections: File, Mode, View, Help, Preferences, and Separators. These six sections can be pressed on to show additional sub-menus with more functions. To start, let's look into the File section.

The File Menu

The File menu contains three selections:Print Session, New Calculator, and Exit; Print session is a way to print a history of all math calculations done during this session. New Calculator will create another window of Fragile with default preferences loaded. When the exit button is pressed, Fragile will close, so be careful!

The Print Session Menu

The print session button opens up a dialog that allows you to print the session history, which stores all of the computations you have executed during the current session.

The new calculator button will open another window of Fragile. This new Fragile window and any existing windows will be completely independent of one another, and operations done on one calculator will not affect any other calculators. The new Fragile window will be loaded with default preferences, but a preference file can be loaded for each window independently.

The Mode Menu

The Mode Menu specifies how results of computations are displayed. When "Proper" is checked, all results will be shown as a proper fraction. When "Reduced" is checked, all fractional results will be in their least possible form.

The View Menu

The View menu also contains a single selection: Pie Chart; When this button is pressed, the Pie Chart menu will open, which will show a representation of the current fraction as a pie chart.

The Help Menu

The Help menu contains two sections: About, which brings up an about page with information on Fragile, and Help, which opens this HTML help page.

The Preferences Menu

The Preference Menu contains three sections: Edit, which allows the setting of keybindings and if the thousands separator should be on or off, Open, which allows you to open a preference file for use inside Fragile, and Save, which allows you to save your currently set preferences for future use. Let's take a closer look at setting and using keybinds.

The edit preference menu allows for keybinds to be set. In this menu, a menu item can be selected from the drop-down menu. Once an item is selected in the drop-down menu, press the button that says "Record." This will cause Fragile to start recording key presses. Simply press record and press a key, and Fragile will set that key to be the shortcut button for the selected menu item. To use the shortcut, press "Alt" and the key you set. You can also set the thousands separator from here.

The open menu allows you to load a previously saved preference file. Simply navigate to the desired file and select it, and Fragile will load the preferences in for use. Please note that you can only use files ending in .fpf.

The save menu allows you to save your currently saved preferences as a .fpf file. Simply select your desired directory, enter a name, and Fragile will save your preferences. Fragile will also save the set preferences under FragileDefaults/defaults.fpf, which can be found in your home directory. These defaults will be loaded whenever Fragile is run, but will not be loaded if the "New Calculator" button is used.

The Separators Menu

The separators menu allows for separators to be added or removed to numbers that are larger that 999. To use them, simply press the "On" button, and Fragile will add the separators to your number to make viewing more easy

This seems like a perfect time to explain the session history and intermediate steps windows. Let's do that now

The Session History Display

When the arrow (>) button is pressed on the right side, the session history display will pop out. This display shows every computation done in this window of Fragile.

The Intermediate Steps Display

When the arrow (<) button is pressed on the left side, the intermediate steps display will pop out. This display shows the steps taken to reach an answer, along with explanations on how that conclusion was reached

Relational Operations

Fragile includes the capability to do relational operations. Let's walk through one now. We will compare 5 2/3 to 5 1/2 and see which one is bigger. Enter "5 2/3" as your first operand, and press the greater than sign (>). This will set Fragile into relational operation mode. Next, enter "5 1/2" and press enter (=). This will open a pop-up window that will show the outcome of the relational operation. In this case, "5 2/3" is greater than "5 1/2".

Fragile includes greater than (>) which evaluates if the first operand is greater than the second, less than (<) which evaluates if the first operand is less than the second, and equal to (≝) which evaluates if the first and second operands are equal.
